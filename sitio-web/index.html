Guía de Dockerización

1. Estructura de archivos

El proyecto tiene esta estructura:

lighttpd-site/
├── index.html
├── css/
│   └── styles.css
└── images/

2. Comando utilizado

docker run -d --name lightserver -p 8080:80 -v "D:\universidad\modulo 3\lighttpd-site":/var/www/html rtsp/lighttpd

Explicación del comando:

  • -d: Ejecuta el contenedor en segundo plano
  • --name lightserver: Asigna un nombre al contenedor
  • -p 8080:80: Mapea el puerto 8080 del host al 80 del contenedor
  • -v "ruta_local":/var/www/html: Monta tu carpeta local en el contenedor

3. Comandos útiles

  • Ver contenedores en ejecución: docker ps
  • Detener el contenedor: docker stop lightserver
  • Iniciar el contenedor: docker start lightserver
  • Eliminar el contenedor: docker rm lightserver
  • Ver logs del contenedor: docker logs lightserver
